# ADR-0002: 两级储位 + FEFO 在补货时控制

## 状态
已采纳

## 背景
客户要求「每个产品有固定储位」（好找、好教、盘点快）。

## 问题
固定储位 = 新到的货倒进同一位 = 批号混堆 → FEFO 作废、追溯断裂。

## 决策
- 拣货位（pick_face）：固定，一 SKU 一位，**同时只能一个批号**
- 存储位（reserve）：动态，按批号分开
- FEFO 在**补货**时控制，不在拣货时控制

Phase 1 建立 `locations.type`；Phase 3 加触发器强制单批号。

## 后果
✅ 拣货员不判断批号  
✅ FEFO 可执行  
❌ 多补货环节；拣货位未清空前不能补新批

## 被否决的方案
- 让拣货员看批号自己挑 → 现实中不会执行
- 放弃固定储位 → 客户不接受
